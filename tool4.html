Install Notes Lib Symbol

How to use Notes Lib Symbol?

  1. Click above button to auto-download the symbol file to your work space.
  2. Insert "Custom Drawing Symbol", browse to folder "c:\workspaces\Symbol", and open "Notes.sym"
    • Tip: click working directory button will return to your workspace quickly.
  3. Switch to "Grouping" page, select proper notes from list
  4. The third page allow you to modify some varaible text. such as relife hole diameter.
  5. Place the Note symbol on your drawing.

How to update my Notes symbol?

  1. Click above button again.
  2. If you can't see updates, try clear all version file of "Notes.sym" in folder: "c:\workspaces\Symbol\".
