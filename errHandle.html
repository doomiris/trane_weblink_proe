Your Pro/E may not be properly configurated to use a Pro/Web.Link tool.

Please Check:

  1. You are NOT accessing previous page through a Pro/E embeded web browser.
  2. Your Pro/E been configured as NOT allowing Pro/Web.Link scripting.
  3. Your Internet security setting allows local scripting and ActiveX.

For new users:

  1. Please download this ZIP file: [SetupScript.zip]
  2. Extract and run the SetupScript.cmd inside.
  3. Restart Pro/E

If you still have trouble on this tool, please contact gplan@irco.com
